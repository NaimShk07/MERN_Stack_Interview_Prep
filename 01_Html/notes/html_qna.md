## 1. ğŸ§± What is HTML?

**Definition:**  
HTML (HyperText Markup Language) is the standard language used to structure and display content on web pages.

**Key Points:**

- HTML defines elements using **tags**.
- It forms the **skeleton** of every website.
- Works with CSS and JavaScript for styling and interactivity.

**Example:**

```html
<!DOCTYPE html>
<html>
	<head>
		<title>My Page</title>
	</head>
	<body>
		<h1>Hello, World!</h1>
	</body>
</html>
```

âœ… Key Points:

- Must start with `<!DOCTYPE html>`.
- Uses nested tags.
- Case-insensitive (but lowercase is preferred).
