# HTML Interview Questions and Answers

## 1. Basic HTML

**Q1: What is HTML and what is its purpose?**  
A: HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It defines the content and structure of a webpage using elements and tags.

**Q2: Difference between HTML, HTML5, and XHTML**

- **HTML**: Original markup language.
- **HTML5**: Latest version of HTML, adds new semantic elements, multimedia support, and APIs.
- **XHTML**: HTML expressed as XML, stricter syntax rules.

**Q3: What are tags, elements, and attributes in HTML?**

- **Tag**: The markup keywords enclosed in `< >`.
- **Element**: The combination of opening tag, content, and closing tag.
- **Attribute**: Provides additional information about an element (e.g., `href`, `src`, `id`).

**Q4: Difference between block-level and inline elements**

- **Block-level**: Start on a new line, take full width (e.g., `<div>`, `<p>`).
- **Inline**: Stay on the same line, only take as much width as needed (e.g., `<span>`, `<a>`).

## 2. HTML Document Structure

**Q5: Essential tags in a basic HTML document**

```html
<!DOCTYPE html>
<html>
	<head>
		<title>Page Title</title>
	</head>
	<body>
		<h1>Hello World</h1>
	</body>
</html>
```

**Q6: Purpose of <head> and <body>**

- <head> : Contains metadata, title, links to CSS, scripts.
- <body> : Contains the visible content of the webpage.

Q7: Purpose of <meta> tag

- Defines metadata like charset, viewport, description.
- Example:

```html
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```
