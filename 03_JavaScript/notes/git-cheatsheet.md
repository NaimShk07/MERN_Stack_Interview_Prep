# Git Cheat Sheet (Interview Focus)

## 1. Clone a Repository

```
git clone <url>
```

## 2. Check Status

```
git status
```

## 3. Stage Files

```
git add .
git add <file>
```

## 4. Commit

```
git commit -m "message"
```

## 5. Push to Remote

```
git push
```

## 6. Pull (fetch + merge)

```
git pull
```

## 7. Fetch (download only)

```
git fetch
```

## 8. Create Branch

```
git branch <name>
```

## 9. Switch Branch

```
git checkout <name>
```

## 10. Create + Switch

```
git checkout -b <name>
```

## 11. Merge Branch

```
git merge <branch>
```

## 12. View Log

```
git log
```
